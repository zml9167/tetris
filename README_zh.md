# 俄罗斯方块游戏

使用 Godot 4.6 引擎实现的经典俄罗斯方块游戏。

## 功能特性

- 经典俄罗斯方块游戏机制
- 键盘控制
- 游戏手柄支持
- 分数系统和等级提升
- 保存/加载功能
- 暂停菜单
- 游戏结束画面
- 最高分记录
- 放弃游戏确认对话框
- 改进的碰撞检测
- 流畅的旋转机制

## 技术栈

- Godot 4.6
- GDScript
- 2D 图形

## 在线游玩

你可以直接在浏览器中游玩：[https://zml9167.github.io/tetris/Tetris.html](https://zml9167.github.io/tetris/Tetris.html)

## 安装方法

1. 下载并安装 [Godot 4.6](https://godotengine.org/download)
2. 克隆本仓库：
   ```bash
   git clone https://github.com/yourusername/tetris.git
   ```
3. 在 Godot 中打开项目，选择 `project.godot` 文件
4. 按播放按钮开始游戏

## 控制方式

### 键盘
- **左箭头/A**：向左移动
- **右箭头/D**：向右移动
- **下箭头/S**：向下移动（软降）
- **上箭头/W/E**：顺时针旋转
- **Q**：逆时针旋转
- **Escape**：暂停游戏

### 游戏手柄
- **方向键左右**：左右移动
- **方向键下**：向下移动
- **A 按钮**：顺时针旋转
- **B 按钮**：逆时针旋转
- **开始按钮**：暂停游戏

## 游戏机制

- 方块从屏幕顶部落下
- 完成整行来获得分数
- 随着等级提升，游戏速度会加快
- 当方块到达屏幕顶部时游戏结束
- 当旋转会导致方块超出底部边界时，旋转会被阻止
- 按住向下箭头可以激活快速下落

## 游戏玩法

1. 在标题画面按任意键开始新游戏
2. 使用控制键移动和旋转下落的方块
3. 尝试完成水平行来获得分数
4. 随着分数增加，游戏速度会加快
5. 按 Escape 键暂停游戏
6. 当方块堆叠到顶部时游戏结束

## 保存和加载

- 游戏会在暂停时自动保存进度
- 你可以从标题画面继续游戏
- 你可以从标题画面放弃当前游戏（需要确认）
